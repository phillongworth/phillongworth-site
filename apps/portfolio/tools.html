Brief description of useful tools for the project

Useful mapping tools

GPS Visualizer

A tool for visualizing GPS data and creating maps from GPX files.

Visit GPS Visualiser

GPX Studio

An online tool for editing and visualizing GPX files with mapping capabilities.

Visit GPX Studio

rowmaps

This site provides rights of way data for 149 authorities in England and Wales have been released with an open licence. The data has been used to contruct the maps on this site.

Visit rowmaps

Leaflet

Leaflet is the leading open-source JavaScript library for mobile-friendly interactive maps.

Visit Leaflet

Segment Explorer

Fantastic site with loads of useful tools for mapping, Strava etc - suchas this one for finding Strava segments, whch wokrs much better than the native Strava tool.

Visit Dashboard

Statshunter

StatsHunters is a website to get more out of your Strava data. It uses your Strava data to show heatmaps and advances statistics. The link is to my stats - use the link on there to set up your own page.

Visit Statshunter

Plotaroute.com

Plotaroute is my favourite tool for planning cycling and walking routes. It allows you to create, share, and view routes on a map, and has access to Ordnance Survey mapping for only £20 per year (as at 2025)

Visit Dashboard
